Центрирование по горизонтали

inline или inline-block элементы внутри родительского можно центрировать так:

	    .wrap {
	      text-align: center;
	    }

Блочные элементы можно центрировать так, задав им свойство:

	    .wrap {
	      margin: 0 auto;
	    }

Любые (блочные, строчные и т.д.) элементы можно центрировать с помощью flexbox, указав родителю:

	    .wrap {
	      display: flex;
	      justify-content: center;
	    }

Центрирование внутри таблиц для inline элементов:

	    .wrap {
	      display: table;
	    }
	    
	    .table-cell {
	      display: table-cell;
	      text-align: center;
	    }

Центрирование по вертикали

Любые (блочные, строчные и т.д.) элементы можно центрировать с помощью flexbox, указав родителю:

	    .wrap {
	      display: flex;
	      justify-content: center;
	      align-items: center;
	    }

С помощью line-height (если текст элемента в одну строку 100%)

	    .wrap {
	      text-align: center;
	    }
	    
	    .item {
	      display: inline-block;
	      line-height: 200px;
	    }

inline, inline-block элементы с помощью элемента призрака (например текста > 1 строки)

	    .wrap {
	      text-align: center;
	      
	      &:before {
	        content: '';
	        display: inline-block;
	        height: 100%;
	        vertical-align: middle;
	      }
	    }
	    
	    .item {
	      display: inline-block;
	      vertical-align: middle;
	    }

Выравнивание по вертикали с помощью таблиц:

	    .wrap {
	      display: table;
	    }
	    
	    .table-cell {
	      display: table-cell;
	      text-align: center;
	      vertical-align: middle;
	    }

Выравние с помощью absolute и т.п. и CSS3

	    .wrap {
	      position: relative;
	    }
	    
	    .item {
	      position: absolute;
	      top: 50%;
	      left: 50%;
	      transform: translate(-50%, -50%);
	    }

Выравние с помощью absolute и т.п. и CSS (если известны размеры элемента)

	    .wrap {
	      position: relative;
	    }
	    
	    .item {
	      position: absolute;
	      top: 50%;
	      left: 50%;
	      transform: translate(-50%, -50%);
	    }
